WatchStormWeb 1.2
Web
WatchStormWeb has officially launched. The functions of adding and deleting movies and TV series, searching for them, as well as a list of favorite movies are available. The website design is aligned with the mobile application design.
WatchStormWeb 1.4
Web
Redesign of over 90% of the website. The "WatchStorm News" section has been added, movie and TV series posters have been replaced with placeholders, and many minor bugs have been fixed.
WatchStorm 1.6
Mobile
Redesign of most of the application. Added the ability to specify a 6-digit code to access WatchStormWeb, updated authorization dialog. Added placeholders for movie and TV series posters. The application is now digitally signed with a certificate. Minor bugs fixed.
WatchStormWeb 1.6
Web
Minor website redesign and many technical changes. Many bugs have been fixed, from minor to critical. Added a settings section, the ability to import and export a list of movies and TV series and many other useful features.
WatchStorm 1.6.2
Mobile
Minor application redesign. Renamed application package, added display of notification when password is changed, added display of notification when setting a 6-digit code. The application is digitally signed with a certificate.
WatchStormWeb 1.6.2
Web
Added WatchStorm Assistant and timeline of WatchStorm versions, as well as indicators in notifications. Changed how the website receives images. Minor website redesign. Minor bugs fixed.
WatchStormWeb 1.6.4
Web
Minor website redesign and complete animations rework. Minor dialogs rework. Many bugs fixed, from minor to critical. Fixed many critical security issues. Added 2FA. Full adaptation of the website for mobile devices. Many minor innovations and fixes.
